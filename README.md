ОПИСАНИЕ ПРОЕКТА

Запуск Angular приложения:
1. cd client
2. npm install
3. ng serve

Запуск JSON-server:
1. cd server
2. npm install
3. npm run server

Было реализовано на клиенте:
1. Верстка по БЭМ методологии в соответствии с макетом и разделение представления на переиспользуемые компоненты
2. Выделены в SCSS константы цветов, типография, а также mixins для изображений, flex и grid контейнеров
3. Реализована возможно выбора диапазона дат при помощи date picker
4. Добавлен interceptor для подставления базовой части url к запросам на сервер
5. Реализован маршрутизация с использованием lazy loading, а также все маршруты выделены в отдельные константы
6. Реализована возможность скачать журнал событий в PDF формате
7. Интегрирован стейт менеджер NgRx для карточек задач и расширенной информации по текущей карточке

На сервере реализовал JSON данные для получения информации для карточек задач, а также для получения расширенной информации по конкретной карточке
